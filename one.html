The features of Html5-

  • Improved Multimedia support - HTML 5 allows for the integration of Multimedia Elements such as audio and video directly into webpages without the need of plugins like flash

  • Canvas Elements - HTML5 introduced the < canvas> element, which allows for dynamic,interactive graphics to be created and manipulated with a Wev page

  • Geolocation API - HTML5 provides an API for obataining the user's location,enabling web application to offer location-based service.

  • Local storage - HTML5 introduced the localStorage API,which allows for the storage of data on the user's device,improving performance and reducing the need for round-tips to the server

  • New Structural Elements - HTML5 introduced new semanatic elements such as <header> ,<footer>,<nav> and <artical> that make it easier to structure and organise content an a webpage.
